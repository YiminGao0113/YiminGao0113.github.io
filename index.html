RISC-V Processor Design using VHDL

This is a single-pipelined processor design using VHDL. The processor can run sequential machine code with the abilities to resolve structure hazards, control hazards and read after write data hazards.

The processor has five stages, a register file, a register tracker and a memory arbiter. The register file consists of 32 32-bit registers and the tracker consists of 32 2-bit registers. The memory arbiter is a totally combinational device for resolving structure hazards.

Improving Rhythm Perception Using Rock Paper Scissors

A group project aiming to improve rhythm perception among people with beat deafness using a rock paper scissor game with a Nao robot. The robot can play rock paper scissors with human with configurable rhythm settings and give feedback when human is not moving on time.

There are four modules in the whole system. Robot movement, human gesture recognition, human rhythm estimation and a web server collecting and computing all the data and giving instructions to the Nao robot.

Chatroom App

This is a simple multi-client chatroom app using sockets in python. The server could either run on localhost or be deployed on a remote server.

The app used multiple threads for data transfering between server side and client side without interrupting each other. The chatroom can refresh all broadcasted messages every 0.1 second.

When the user enters the word quit, the chatroom will exit and the user will be disconnected from the server.
